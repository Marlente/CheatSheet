Personal Cheatsheet
HTML general setup
Tag Description
<html> Tells the browser this is HMTL code
<head> Meta description including title, links, stylesheets
<body> Container for body / main content of the website
<nav> Primary and secundary menus to link the page
<footer> Conatct, Impressum, Links, Copyright
<!DOCTYPE HTML> Opening statement of the file

Code Example for the general setup

<!DOCTYPE html> <html lang="en"> <head> <title>Title<title> </head> <body> <header> Logo, title, navigation <nav> Primary and secondary menus but ever nest navs </nav> </header> <section> Related grouping of semantic meaning <article> Self-contained content </article> <article> Would still make sense if all other content removed <aside> <blockquote> Related to main but outside the main flow </blockquote> </aside> </article> </section> <footer> Copyright, contact, links </footer> </body> </html>
HTML tables
Tag Name Description
<table> Table Container for the Table
<thead> Header Rows containing header for colums
<tbody> Body Container for table body
<tr> Row Container for table row
<td> Data Container for data in a row
<colspan> Span How many columns are taken by one td element (attribute)

Code Example for setup

<table> <caption>Table Caption</caption> <colgroup> <col> <col span="2" class="left-two"> <col span="2" class="right-two"> </colgroup> <thead> <tr> <th></th> <th colspan="2">Left Two</th> <th colspan="2">Right Two</th> </tr> </thead> <tbody> <tr> <th rowspan="2">Top Two</th> <td>Body 1,1</td> <td>Body 1,2</td> <td>Body 1,3</td> <td>Body 1,4</td> </tr> <tr> <td>Body 2,1</td> <td>Body 2,2</td> <td>Body 2,3</td> <td>Body 2,4</td> </tr> <tr> <th rowspan="2">Bottom Two</th> <td>Body 3,1</td> <td>Body 3,2</td> <td>Body 3,3</td> <td>Body 3,4</td> </tr> <tr> <td>Body 4,1</td> <td>Body 4,2</td> <td>Body 4,3</td> <td>Body 4,4</td> </tr> </tbody> <tfoot> <tr> <th></th> <td>Footer 1</td> <td>Footer 2</td> <td>Footer 3</td> <td>Footer 4</td> </tr> </tfoot> </table>
